Envia automaticamente para o Twitter os posts publicados no blog, com suporte a posts agendados.

== Description ==

O Twitting adverte em sua conta no Twitter quando existem novas publicações.
O Twitting é a continuação do Tweet myPost, só que com suporte apenas em português.

== Installation ==

1. Envie a pasta `twitting` para o diretório `/wp-content/plugins/`
2. Ative o plugin indo no menu 'Plugins' na dashboard do WordPress
3. Clique no menu 'Twitting' e configure.
